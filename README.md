# GestaoDeEncomendasArtesanais

Projeto feito por: 
Emily Dimas Candido

O tema do trabalho: 
gestão artesanal (podendo criar pedidos e produtos)

O problema resolvido: 
automação de encomendas para pessoas que fazem trabalhos artesanais 
em organização de pedidos e produtos

Possuí duas entidades: 
pedido e produto

Instruções de execução:
O sistema deve ser rodado com o comando gradle run pelo terminal, as telas são super intuítivas!
